THEME: Educational Value (GST Mistakes)
ATTACHMENT: Carousel image or infographic showing the 3 mistakes
================================================================

3 GST mistakes Indian freelancers make that can get their invoices rejected:

Mistake 1: Wrong tax type

Same state client → CGST (9%) + SGST (9%)
Different state client → IGST (18%)

Most freelancers apply 18% flat every time.
That's wrong — and it creates reconciliation issues for your client.

Mistake 2: Missing or wrong HSN code

Every service has a specific HSN/SAC code.
It's mandatory on invoices above ₹5,000.

Design services → 998314
Development → 998313
Consulting → 998231
Content writing → 998390

Mistake 3: No "Place of Supply" mentioned

This field determines which tax applies.
Without it, your invoice is incomplete under GST law.

I've been working on a tool that handles all of this automatically.

More on that soon.

Save this post for the next time you raise an invoice.
And share it with a freelancer friend who needs it. 🙏
